.. GMT Modern Theme

Selections for modern theme
================================================================================

Please see `this pull request <https://github.com/GenericMappingTools/gmt/pull/3344>`_ and 
`this GitHub issue <https://github.com/GenericMappingTools/gmt/issues/2290>`_ for information
about the selection of settings for modern theme. The theme settings pages (links above) show Mercator, Lambert Conic Conformal, and Cartesian
plots in multiple sizes for the following configurations:
